快速方法

推荐

通过 PowerShell 自动下载并运行。

  1. 1以管理员权限打开 PowerShell
  2. 2粘贴下方命令并回车
  3. 3等待下载，按说明操作
& ([scriptblock]::Create((irm "https://debloat.raphi.re/")))

传统方法

手动下载 ZIP 并运行脚本。

  1. 1下载最新版本 并解压 ZIP
  2. 2进入 Win11Debloat 文件夹
  3. 3双击 Run.bat
  4. 4同意 UAC 提示，按说明操作

高级方法

手动 PowerShell 执行，支持命令行参数。

  1. 1下载并解压至指定路径
  2. 2管理员 PowerShell 中执行：
Set-ExecutionPolicy Unrestricted -Scope Process -Force
.\Win11Debloat.ps1

功能概览

全面掌控您的系统

覆盖隐私、性能、界面每个角落。所有更改均可轻松还原。

默认设置预设

应用移除

移除各种预装垃圾应用 (Bloatware)。可通过 Microsoft Store 重新安装。详情 →

隐私保护

  • • 禁用遥测、诊断数据、活动历史、定向广告
  • • 禁用 Windows 各处的提示、建议和广告
  • • 禁用定位服务和"查找我的设备"
  • • 禁用锁屏/桌面的 Windows 聚焦
  • • 禁用 Edge 中的广告和 MSN 新闻提要
  • • 隐藏设置中的 Microsoft 365 广告

AI 功能

  • • 禁用并移除 Copilot / Recall (回忆) / Click to Do
  • • 阻止 AI 服务 (WSAIFabricSvc) 自动启动
  • • 禁用 Edge、画图、记事本中的 AI 功能

系统设置

  • • 恢复旧版 Win10 右键菜单
  • • 关闭鼠标加速 / 禁用粘滞键快捷键
  • • 禁用存储感知自动磁盘清理
  • • 禁用快速启动以确保完全关机
  • • 禁用 BitLocker 自动加密
  • • 禁用拖拽托盘 / 现代待机网络连接

Windows 更新

  • • 阻止立即获取更新 / 阻止更新后自动重启
  • • 禁用传递优化（P2P 共享已下载更新）

外观

  • • 启用深色模式 / 禁用透明效果 / 禁用动画和视觉效果

开始菜单
与搜索

  • • 移除或替换开始菜单固定应用
  • • 隐藏"推荐"和"所有应用"部分
  • • 禁用手机连接集成
  • • 禁用 Bing 搜索和 Copilot 集成
  • • 禁用 Store 应用建议 / 搜索亮点 / 搜索历史

任务栏

  • • 图标向左对齐 / 隐藏搜索框
  • • 隐藏任务视图、聊天图标、小组件
  • • 启用右键"结束任务"
  • • 启用"最后一次活动点击"行为
  • • 自定义多显示器图标和按钮合并模式

资源管理器

  • • 更改默认打开位置
  • • 显示文件扩展名和隐藏文件
  • • 隐藏导航窗格中"主页"/"图库"
  • • 隐藏重复的可移动驱动器条目
  • • 将常用文件夹添加至"此电脑"
  • • 清理右键菜单冗余选项

多任务处理

  • • 禁用窗口贴靠及辅助/布局建议
  • • 更改 Alt+Tab 是否显示浏览器标签页

可选功能
与其他

  • • 启用 Windows 沙盒 / WSL
  • • 禁用 Xbox Game Bar 和屏幕录制
  • • 禁用 Brave 中的冗余功能

高级功能

  • • 对其他用户应用更改，而非仅限当前用户
  • • Sysprep 模式：应用到 Windows 默认用户配置文件

所有更改均可还原

几乎所有应用都可以通过 Microsoft Store 重新安装。完整还原指南已就绪。
